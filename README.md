## Getting Started :


### Prerequisites 
* VS Code


### Installation 
* Clone the repository
    ``` 
  git clone https://github.com/pankaj5417/adminDashboard.git
    ```
To run on local server
### open Terminal for client
 * cd/client 
 * npm i
 * npm start
 
 ### open another terminal for server
 * cd/server
 * npm i
 * npm start
